# 개인정보 처리방침

**최종 업데이트:** 2025년 11월 30일

## 1. 개요

니코마(Nico-ma)는 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 이 개인정보 처리방침은 니코마가 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

## 2. 수집하는 정보

### 2.1 자동으로 수집되는 정보

니코마는 **어떠한 개인정보도 자동으로 수집하지 않습니다.**

### 2.2 사용자가 입력하는 정보

사용자가 직접 입력하는 다음 정보만 저장됩니다:

- 저장한 상품 정보 (상품명, 가격, URL, 카테고리)
- 사용자가 추가한 카테고리 이름
- 사용자가 등록한 검색어
- 사용자가 추가한 즐겨찾기 사이트
- 사용자 설정 (테마 선택 등)

## 3. 정보 저장 위치

### 3.1 로컬 저장소

모든 데이터는 **사용자의 브라우저 로컬 저장소**에만 저장됩니다.

- Chrome의 `chrome.storage.local` API 사용
- 사용자의 컴퓨터에만 존재
- **외부 서버로 전송되지 않음**

### 3.2 크롬 동기화 (선택사항)

사용자가 크롬 동기화를 활성화한 경우:
- Google 계정을 통해 데이터 동기화 가능
- Google의 개인정보 처리방침 적용
- 동기화 끄기 가능

## 4. 정보 사용 목적

수집된 정보는 오직 다음 목적으로만 사용됩니다:

- ✅ 저장한 상품 목록 표시
- ✅ 가격 계산 기능 제공
- ✅ 카테고리별 정리
- ✅ 사용자 설정 유지

**다음 목적으로는 절대 사용되지 않습니다:**
- ❌ 광고 목적
- ❌ 제3자 공유
- ❌ 사용자 추적
- ❌ 데이터 판매

## 5. 정보 공유

니코마는 **어떠한 제3자와도 사용자 정보를 공유하지 않습니다.**

## 6. 외부 서비스

### 6.1 환율 정보

실시간 환율 정보를 위해 외부 API를 사용할 수 있습니다:
- 환율 정보만 요청
- 사용자 개인정보는 전송되지 않음

### 6.2 번역 서비스

검색어 번역 기능을 위해 외부 번역 API를 사용할 수 있습니다:
- 검색어 텍스트만 전송
- 사용자 식별 정보는 전송되지 않음

## 7. 데이터 보안

### 7.1 로컬 저장

- 모든 데이터는 브라우저의 보안 저장소에 암호화되어 저장
- Chrome의 샌드박스 환경에서 보호

### 7.2 백업 파일

- 사용자가 직접 생성한 백업 파일(JSON)
- 사용자가 직접 관리
- 민감한 정보 포함 시 안전하게 보관 권장

## 8. 사용자 권리

### 8.1 데이터 열람

사용자는 언제든지 자신의 데이터를 확인할 수 있습니다:
- 확장 프로그램 팝업에서 모든 정보 확인 가능

### 8.2 데이터 수정

사용자는 언제든지 데이터를 수정할 수 있습니다:
- 상품, 카테고리, 검색어 편집 가능

### 8.3 데이터 삭제

사용자는 언제든지 데이터를 삭제할 수 있습니다:
- 개별 항목 삭제
- 확장 프로그램 제거 시 모든 데이터 삭제

### 8.4 데이터 내보내기

사용자는 언제든지 데이터를 내보낼 수 있습니다:
- JSON 형식으로 다운로드 가능
- 다른 기기로 이동 가능

## 11. 권한 설명

니코마가 요청하는 Chrome 권한과 사용 목적:

| 권한 | 사용 목적 |
|------|----------|

| `activeTab` | 현재 페이지의 상품 정보 읽기 |
| `tabs` | 즐겨찾기 사이트를 새 탭에서 열기 |
| `contextMenus` | 우클릭 메뉴 제공 |
| `sidePanel` | 사이드패널 기능 제공 |
