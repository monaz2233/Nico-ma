# 🛒 Nico-ma (니코마)

일본 쇼핑몰 상품을 카테고리별로 저장하고 가격 계산까지! 일본 쇼핑의 필수 도구

[![Version](https://img.shields.io/badge/version-1.3.2-green)](https://github.com/yourusername/nicoma/releases)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

---

## ✨ 주요 기능

### 📦 상품 저장 및 관리
- 클릭 한 번으로 상품 정보 자동 저장 (이름, 가격, URL)
- 카테고리별로 깔끔하게 정리
- 태그 시스템으로 세부 분류
- 카테고리별 상품 목록 보기

### 💰 가격 계산기
- 여러 상품의 총 가격 자동 계산
- 실시간 엔화 → 원화 환율 적용
- 카테고리별 필터링
- 체크박스로 간편한 선택

### 🔍 빠른 검색어 복사
- 상단 고정 검색어 바
- 클릭 한 번에 일본어 복사
- 한국어 → 일본어 자동 번역
- 최대 6개 검색어 등록 가능

### ⭐ 즐겨찾기
- 자주 가는 쇼핑몰 빠른 접근
- 실제 파비콘 표시
- 새 탭에서 바로 열기

### 🎨 5가지 테마
- ☀️ 라이트 - 깔끔한 화이트
- 🌙 다크 - 편안한 블랙
- 💙 오션 - 시원한 블루
- 🌸 핑크 - 부드러운 핑크
- 🍃 민트 - 상쾌한 그린

### 💾 안전한 데이터 관리
- JSON 형식 백업/복원
- 크롬 동기화 지원
- 데이터 내보내기로 안전 보관

### 🌍 모든 웹사이트 지원

니코마는 **모든 웹사이트**에서 작동합니다!
- 위 목록에 없는 사이트도 수동으로 정보 입력 가능
- 상품 URL은 자동 저장
- 가격은 수동 입력 또는 자동 인식

** 지원이 안되는 사이트가 있다면 이메일로 연락 부탁드립니다 **

## 🚀 사용 방법

### 1️⃣ 설치
[Chrome Web Store에서 설치](chrome-web-store-링크)

### 2️⃣ 상품 저장하기
1. 일본 쇼핑몰 상품 페이지 방문
2. 니코마 아이콘(🛒) 클릭
3. 카테고리 선택
4. "저장" 버튼 클릭

### 3️⃣ 검색어 활용
1. 상단 검색어 바에서 클릭
2. 자동으로 일본어 복사
3. 쇼핑몰 검색창에 붙여넣기

### 4️⃣ 가격 계산
1. 사이드패널 열기 (아이콘 우클릭 → "사이드 패널 열기")
2. "계산기" 탭 선택
3. 원하는 상품 체크
4. 총 가격 자동 계산

## 🛠️ 기술 스택

- **언어:** HTML, CSS, JavaScript
- **플랫폼:** Chrome Extension (Manifest V3)
- **권한:**
  - `storage` - 데이터 저장
  - `activeTab` - 현재 페이지 정보
  - `tabs` - 탭 관리
  - `contextMenus` - 우클릭 메뉴
  - `sidePanel` - 사이드패널
